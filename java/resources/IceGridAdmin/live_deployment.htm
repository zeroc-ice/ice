Live Deployment

The Live Deployment pane shows the runtime status and configuration of an existing IceGrid deployment, and allows you to perform various administrative tasks on this deployment, such as:

•start or stop a server
•enable or disable a server
•send signal to a server
•retrieve the log files of a server
•patch an entire application
•register a new "well-known" object with the IceGrid registry
•undeploy a deployed application
