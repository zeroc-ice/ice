Database Environment

A Database Environment represents a Berkeley DB database environment, typically used for storage by the Freeze persistence service.

Properties

The Database Environment Properties panel shows:

Description

A free-text description of this Berkeley DB database environment.

DB Home

The path to the home directory of this database environment. Corresponds to the Ice property Freeze.DbEnv.env-name.DbHome. It is often created by the IceGrid node in the server's private directory.

Properties

Berkeley DB configuration properties for this database environment. IceGrid generates a DB_CONFIG file in the DB Home directory with these properties.
