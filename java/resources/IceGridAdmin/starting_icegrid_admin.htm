Starting IceGrid Admin

In some environments, IceGrid Admin can be started by simply double-clicking on the IceGridGUI.jar file.

On all platforms, you can start IceGrid Admin from a terminal by typing:

java -jar path-to-IceGridGUI.jar

For example on a Linux system with an Ice RPM installation:

$ java -jar /usr/lib/Ice-3.2.0/IceGridGUI.jar
