System Requirements

IceGrid Admin is a JavaTM application supported a wide range of platforms including Windows XP, MacOS X and Linux.

In order to run IceGrid Admin you need:

•the IceGridGUI.jar file, usually installed in the bin or lib directory of your Ice installation. If you are using a Linux RPM installation, IceGridGUI.jar is installed in /usr/lib/Ice-3.2.0.
•a Java Runtime Environment (JRE) v1.4.2 or later (1.5.0 or later is recommended). For Windows, Linux and Solaris, we recommend to use a Sun JRE or JDK, available here.

 

If you want to read IceGrid XML files from IceGrid Admin, you also need to have the icegridadmin command-line utility (version 3.2.0) in your PATH.
