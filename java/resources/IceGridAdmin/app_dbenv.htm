Database Environment

Properties

The Database Environment Properties panel offers the following fields:

Name

The name of the Freeze Database Environment. This name must be unique within the enclosing Ice communicator.

Description

A free-text description of this database environment.

DB Home

The path of the home directory of the Berkeley DB environment. The default is "Created by the IceGrid Node": when selected, IceGrid node creates automatically a directory in its local server tree. Otherwise, if you enter your own value, the path must exist: IceGrid node will not create it if it does not.
