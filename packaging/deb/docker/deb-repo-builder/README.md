# ZeroC Ice DEB Repo Builder

This Docker image provides a minimal, debian  environment for:

- Generating a DEB package repository

## Image Name

`ghcr.io/zeroc-ice/deb-repo-builder-<channel>`

Where `<channel>` is the Ice version channel (e.g., `3.8`, `3.9`).

## Included Tools

- reprepro
- gnupg2
